Nullable / absent value

An absent value represents the idea that data might be missing instead of always being present.
Some languages use a dedicated option type, while others use a special null-like value that requires extra care.
In Java, the traditional absent value is null, and modern code often uses Optional to make absence more explicit.

// Java
import java.util.Optional;

Optional<String> nickname = Optional.ofNullable(null);

String value = nickname.orElse("anonymous");
System.out.println(value); // anonymous
