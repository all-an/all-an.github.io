Loop / iteration

Iteration repeats a block of code while a condition holds or while elements remain in a sequence.
Common forms include for, while, and collection-based loops, each trading control for readability and convenience.
It is the standard way to process repeated work without manually duplicating the same statements.

// Java
for (int i = 0; i < 3; i++) {
  System.out.println("step " + i);
}
