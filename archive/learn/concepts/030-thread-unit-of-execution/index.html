Thread / unit of execution

A thread is an independent path of execution that can run work concurrently with other threads.
Java has supported threads for a long time, and modern Java also adds virtual threads for lighter-weight tasks.
The main idea is that multiple flows of control can progress at the same time.

// Java
Thread worker = Thread.startVirtualThread(() -> {
  System.out.println("running on " + Thread.currentThread());
});

try {
  worker.join();
} catch (InterruptedException ignored) {
}
System.out.println("done");
