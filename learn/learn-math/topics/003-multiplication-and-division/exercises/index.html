01 Multiply two numbers

Write multiply(a, b) that returns the product of a and b using the * operator.

return a * b;
02 Multiply as repeated addition

Write mulRepeat(a, b) that computes a × b using only addition — no * operator. Assume b is a non-negative integer.

Start with let result = 0; then for (let i = 0; i < b; i++) result += a;
03 Safe division

Write safeDivide(a, b) that returns a / b as a float. If b === 0 return null instead of dividing.

Check if (b === 0) return null; before dividing. Otherwise return a / b; — JS division always gives a float.
04 Factorial

Write factorial(n) that returns n! for n ≥ 0. Recall: 0! = 1 and n! = n × (n−1) × … × 1. Do not use any library factorial.

Start with let result = 1; then for (let i = 2; i <= n; i++) result *= i;
05 Is divisible?

Write isDivisible(n, d) that returns true if d divides n exactly (remainder 0), and false otherwise. Return false if d === 0.

Guard with if (d === 0) return false; then return n % d === 0;
06 GCD — Euclidean algorithm

Implement gcd(a, b) using the Euclidean algorithm. Do not use any built-in GCD. The algorithm: while b !== 0, replace (a, b) with (b, a % b); return a.

while (b !== 0) { [a, b] = [b, a % b]; } then return a;
07 Times table row

Write timesTable(n) that returns an array of the first 10 multiples of n: [n×1, n×2, …, n×10].

for (let i = 1; i <= 10; i++) result.push(n * i);
08 Prime factorization

Write primeFactors(n) that returns the prime factorization of n as a sorted array (with repeats). E.g. primeFactors(60) → [2, 2, 3, 5]. Assume n ≥ 2.

Start with divisor let d = 2;. While d * d <= n, if n % d === 0 push d and set n = Math.floor(n / d);. Otherwise d++;. If n > 1 at the end, push it.
