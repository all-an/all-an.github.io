Multiplication & Division

Multiplication is repeated addition. Division is its inverse — splitting a quantity into equal groups. Together they unlock factors, multiples, primes, and the building blocks of number theory.

Multiplication

What is Multiplication?

Multiplying a by b means adding a to itself b times. The result is the product; the inputs are factors.

a × b = a + a + … + a  (b times)
// example
4 × 3 = 4 + 4 + 4 = 12
Commutative

a × b = b × a

Associative

(a×b)×c = a×(b×c)

Distributive

a×(b+c) = a×b + a×c

Identity

a × 1 = a

Zero property

a × 0 = 0

Closed in ℕ

Product of two naturals is a natural.

In JavaScript

js
// basic multiplication
const a = 4, b = 3;
console.log(a * b);           // 12

// multiplication as repeated addition
function mulRepeat(a, b) {
    let result = 0;
    for (let i = 0; i < b; i++) result += a;
    return result;
}

console.log(mulRepeat(4, 3));  // 12

// distributive: a*(b+c) === a*b + a*c
const c = 5;
console.log(a * (b + c) === a*b + a*c);  // true

Division

What is Division?

Division asks: how many times does the divisor fit into the dividend? The answer is the quotient. It is the inverse of multiplication.

a ÷ b = q  ⟺  q × b = a
// example
12 ÷ 4 = 3  ⟺  3 × 4 = 12
Not commutative

a ÷ b ≠ b ÷ a in general.

Not associative

(a÷b)÷c ≠ a÷(b÷c) in general.

Division by zero

a ÷ 0 is undefined.

Not closed in ℕ

7 ÷ 2 = 3.5 ∉ ℕ. Need ℚ.

In JavaScript

js
// true division → float if needed
console.log(12 / 4);   // 3
console.log(7 / 2);    // 3.5

// floor division → Math.floor rounds toward −∞
console.log(Math.floor(7 / 2));    // 3
console.log(Math.floor(-7 / 2));   // -4  (floors toward -∞)

// modulo → remainder
console.log(7 % 2);    // 1
console.log(10 % 3);   // 1

// quotient and remainder together
const [q, r] = [Math.floor(17 / 5), 17 % 5];
console.log(q, r);     // 3 2  →  17 = 3×5 + 2

// division by zero → Infinity in JS (no exception)
console.log(1 / 0);    // Infinity
Floor Division & Modulo

The Division Algorithm

For any integers a and b (b ≠ 0) there exist unique integers q and r with 0 ≤ r < |b| such that:

a = q × b + r
// example: 17 = 3 × 5 + 2

JavaScript: Math.floor(a/b) gives q, % gives r. Modulo is everywhere — even/odd, wrapping, hashing, cryptography.

Common modulo patterns

js
// even / odd
function isEven(n) { return n % 2 === 0; }
console.log(isEven(4));   // true
console.log(isEven(7));   // false

// clock arithmetic
function nextHour(h) { return (h + 1) % 12; }
console.log(nextHour(11));  // 0

// divisibility check
function divides(d, n) { return n % d === 0; }
console.log(divides(3, 12));  // true
Factors & Multiples

Factors, Multiples & Primes

A factor of n divides it with remainder 0. A prime has exactly two factors: 1 and itself. Every integer > 1 factors uniquely into primes (Fundamental Theorem of Arithmetic).

js
function factors(n) {
    return Array.from({length: n}, (_, i) => i + 1).filter(i => n % i === 0);
}

console.log(factors(12));  // [1, 2, 3, 4, 6, 12]

function isPrime(n) {
    if (n < 2) return false;
    for (let i = 2; i <= Math.sqrt(n); i++)
        if (n % i === 0) return false;
    return true;
}

console.log(isPrime(13));  // true
console.log(isPrime(15));  // false  (3 × 5)

function primeFactors(n) {
    const result = [];
    let d = 2;
    while (d * d <= n) {
        while (n % d === 0) { result.push(d); n = Math.floor(n / d); }
        d++;
    }
    if (n > 1) result.push(n);
    return result;
}

console.log(primeFactors(60));  // [2, 2, 3, 5]
GCD & LCM

Greatest Common Divisor & Least Common Multiple

The GCD is the largest shared factor. The LCM is the smallest shared multiple. Identity: GCD(a,b) × LCM(a,b) = a × b.

js
// Euclidean algorithm
function myGcd(a, b) {
    while (b) { [a, b] = [b, a % b]; }
    return a;
}

console.log(myGcd(48, 18));  // 6

function lcm(a, b) {
    return a * b / myGcd(a, b);
}

console.log(lcm(4, 6));   // 12
console.log(lcm(3, 7));   // 21
Euclidean algorithm (≈300 BC): repeatedly replaces (a, b) with (b, a mod b) until b = 0. The last non-zero a is the GCD. Runs in O(log min(a,b)) steps.

Summary

Key Takeaways

a × b = b × a

Multiplication is commutative. Division is not.

a = q×b + r

Division algorithm. JS: Math.floor(a/b) for q, % for r.

Unique prime factorization

Every integer > 1 = unique product of primes.

GCD × LCM = a × b

Euclidean algorithm finds GCD in O(log n).
