04 Gauss formula

Write sumToN(n) that returns the sum of all integers from 1 to n inclusive using the closed-form formula: n × (n + 1) / 2. Return an integer (use Math.trunc or integer-safe arithmetic).

return Math.trunc(n * (n + 1) / 2); — the product n*(n+1) is always even, so integer division is exact.
05 Absolute difference

Write absDiff(a, b) that returns the absolute (non-negative) difference between a and b. Do not use Math.abs — implement the logic yourself with an if statement.

Compute const d = a - b; then return d >= 0 ? d : -d;
06 Running balance

Write runningBalance(transactions) that takes an array of numbers (positive = deposit, negative = withdrawal) and returns an array of the cumulative balance after each transaction, starting from 0.

Keep a let balance = 0; variable. Use for (const t of transactions), add each to balance, then result.push(balance);

08 Column addition with carry

Write columnAdd(a, b) that adds two non-negative integers digit by digit with carry — like the paper algorithm — without using + on the full numbers. Extract digits with % 10 and drop them with Math.trunc(x / 10).

Loop while a > 0 || b > 0 || carry > 0. Each step: const col = (a % 10) + (b % 10) + carry; push col % 10 to digits, set carry = Math.trunc(col / 10); and drop digits with a = Math.trunc(a / 10); Finally reconstruct from digits in reverse.
