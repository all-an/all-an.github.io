01 Simplify a fraction

Write simplify(num, den) that returns the fraction in simplest form as a two-element array [num, den]. Use the GCD to reduce. E.g. simplify(6, 4) → [3, 2].

First write a gcd(a, b) helper using the Euclidean algorithm, then divide both num and den by gcd(Math.abs(num), Math.abs(den)).
02 Add two fractions

Write addFractions(n1, d1, n2, d2) that adds the fractions n1/d1 + n2/d2 and returns the result as a simplified [num, den] array.

Cross-multiply: num = n1 * d2 + n2 * d1, den = d1 * d2, then call simplify.
03 Fraction to decimal

Write toDecimal(num, den) that returns the decimal value of the fraction num / den as a JavaScript number.

Simply return num / den; — JavaScript division always produces a float when the result is not an integer.
04 Is it a proper fraction?

Write isProper(num, den) that returns true if num/den is a proper fraction (absolute value of numerator is less than the denominator), and false otherwise.

return Math.abs(num) < den;
05 Convert to mixed number

Write toMixed(num, den) that converts an improper fraction to a mixed number. Return a three-element array [whole, remainderNum, den]. E.g. toMixed(7, 2) → [3, 1, 2] (= 3½).

const whole = Math.floor(num / den); const rem = num % den; then return [whole, rem, den].
06 Multiply two fractions

Write multiplyFractions(n1, d1, n2, d2) that multiplies the two fractions and returns the result as a simplified [num, den] array.

Multiply numerators together and denominators together: simplify(n1 * n2, d1 * d2).
07 Round to decimal places

Write roundTo(n, places) that rounds the number n to places decimal places and returns the result. E.g. roundTo(3.14159, 2) → 3.14.

const factor = 10 ** places; then return Math.round(n * factor) / factor;
08 Fraction to percentage

Write fractionToPercent(num, den) that converts the fraction to a percentage value (0–100). E.g. fractionToPercent(3, 4) → 75.

return (num / den) * 100;
