## Compile Kotlin to Javascript:

```bash
.\compile-kotlin.bat
```

## Run Locally

**With VS Code Live Server:**
1. Install "Live Server" extension
2. Right-click `index.html` → "Open with Live Server"

**With npm:**
```bash
npx live-server
```

**With Python:**
```bash
python -m http.server 8000
```

Then open `http://localhost:8000`
