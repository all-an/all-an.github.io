Conditional / branching

Conditional branching chooses between different execution paths based on whether an expression evaluates to true or false.
The classic form is if / else, but many languages also add expression-based variants like when, case, or cond.
It is one of the core building blocks of control flow, turning boolean logic into concrete program behavior.

// Java
int age = 20;
if (age >= 18) {
  System.out.println("adult");
} else {
  System.out.println("minor");
}
