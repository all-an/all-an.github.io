Inheritance / subclassing

Inheritance lets one class reuse and extend the behavior of another, creating a subtype relationship between them.
A subclass inherits fields and methods from its parent and can add new members or override existing behavior.
In Java, subclassing uses extends, and overridden methods allow polymorphic behavior through the parent type.

// Java
class Animal {
  void speak() {
    System.out.println("some sound");
  }
}

class Dog extends Animal {
  @Override
  void speak() {
    System.out.println("woof");
  }
}

Animal pet = new Dog();
pet.speak(); // woof
