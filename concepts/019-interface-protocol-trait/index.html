Interface / protocol / trait

An interface describes a set of operations a type must provide without fixing the concrete implementation.
It lets different types share the same contract, which is useful for substitution and polymorphism.
In Java, interfaces define methods that classes can implement, and code can depend on the interface instead of a specific class.

// Java
interface Greeter {
  String greet();
}

class EnglishGreeter implements Greeter {
  public String greet() {
    return "Hello";
  }
}

Greeter greeter = new EnglishGreeter();
System.out.println(greeter.greet()); // Hello
