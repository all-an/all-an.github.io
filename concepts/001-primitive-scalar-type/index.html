Primitive / scalar type

A primitive type holds a single, fixed-size value directly in memory — no methods, no identity, just raw data.
Java has 8 built-in primitives: boolean, byte, char, short, int, long, float, and double.
They are stack-allocated and cannot be null, making them faster and cheaper than their object wrappers.

int age        = 30;    // 32-bit signed integer
double price   = 9.99;  // 64-bit floating point
boolean active = true;  // true or false
char grade     = 'A';   // single UTF-16 character
