Ternary / inline conditional

A ternary or inline conditional chooses between two values inside a single expression instead of a multi-line if / else block.
It is useful when the decision is small and the result should be assigned or returned directly where it is computed.
Used well, it keeps code compact; overused, it becomes harder to read than a normal branch.

// Java
int age = 20;
String label = age >= 18 ? "adult" : "minor";
System.out.println(label);
