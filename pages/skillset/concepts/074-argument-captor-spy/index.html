Argument captor / spy

Two Mockito tools for behavior verification — checking how a collaborator was used.
An ArgumentCaptor grabs the actual argument passed to a mock so you can assert on its fields. It earns its keep when the code under test builds the argument internally: here AccountService.withdraw constructs an AuditEntry the test never sees, so capturing it is the only way to inspect what was recorded.
A spy wraps a real object: its real methods run by default (the entry is actually stored), while Mockito still records every call so verify and capture work. That is the difference from a mock — a mock is hollow, a spy is a real object with recording bolted on. This is a real Maven project that runs JUnit 5 + Mockito; see README.md.

// Java + Mockito + JUnit 5
// ArgumentCaptor: grab the AuditEntry the service built internally, then assert
// on its fields — impossible with a plain verify(eq(...)) when the test never
// constructs the object being passed.
@Test
void capturesTheAuditEntryArgument() {
  AuditLog auditLog = mock(AuditLog.class);
  AccountService service = new AccountService(auditLog);

  service.withdraw("acc-1", 2500);

  ArgumentCaptor<AuditEntry> captor = ArgumentCaptor.forClass(AuditEntry.class);
  verify(auditLog).record(captor.capture());

  AuditEntry captured = captor.getValue();
  assertEquals("acc-1", captured.account());
  assertEquals(2500, captured.amountCents());
  assertEquals("WITHDRAW", captured.action());
}

// Spy: wrap a REAL InMemoryAuditLog. The real record() runs (the entry is truly
// stored), AND the call is recorded so verify + capture still work.
@Test
void spyRunsRealCodeAndStillRecordsInteractions() {
  InMemoryAuditLog real = new InMemoryAuditLog();
  AuditLog spy = spy(real);
  new AccountService(spy).withdraw("acc-2", 99);

  // Real method actually ran: the entry is in the real backing list.
  assertEquals(1, real.entries().size());

  // And the interaction was still recorded.
  ArgumentCaptor<AuditEntry> captor = ArgumentCaptor.forClass(AuditEntry.class);
  verify(spy).record(captor.capture());
  assertEquals(99, captor.getValue().amountCents());
}
