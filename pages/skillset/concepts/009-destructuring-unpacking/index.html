Destructuring / unpacking

Destructuring takes a compound value like an array or object and pulls its parts into separate variables in a single step.
It makes code shorter when you only need a few fields or positions instead of the whole structure.
JavaScript supports destructuring for both arrays and objects, including renaming and default values.

// JavaScript
const point = { x: 4, y: 7 };
const [first, second] = ["red", "blue"];
const { x, y } = point;

console.log(first);  // "red"
console.log(x, y);   // 4 7
