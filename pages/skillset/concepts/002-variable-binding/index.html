Variable / binding

A variable is a named binding that associates an identifier with a value stored in memory.
In Java, every variable has a declared type and must be initialized before use — the compiler enforces this.
var (Java 10+) lets the compiler infer the type locally, reducing verbosity without losing static typing.

// Java
int score = 100;           // explicit type
var name  = "Allan";       // type inferred as String
final double PI = 3.14159; // binding that cannot be reassigned
