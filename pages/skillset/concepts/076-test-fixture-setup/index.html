Test fixture / setup

A test fixture is the fixed, known starting state a test runs against. Tests must be isolated — the result of one cannot depend on another having run first — so the fixture is rebuilt fresh for each test rather than shared. In JUnit 5 that rebuild lives in a @BeforeEach method, which the runner calls before every @Test; @AfterEach tears it back down. The once-per-class pair @BeforeAll / @AfterAll handle expensive shared setup.
The same idea appears everywhere: setUp (Python unittest), beforeEach (Jest), before(:each) (RSpec), TestInitialize (C#), ScalaTest fixtures. It is one of Kent Beck's original xUnit patterns. This is a runnable Maven project; see README.md.

// Java + JUnit 5
// The fixture is a fresh ShoppingCart, rebuilt before EVERY test by @BeforeEach,
// so the two tests below cannot leak state into each other.
class ShoppingCartTest {

  // The fixture: the object under test, recreated for each test method.
  private ShoppingCart cart;

  // Fixture setup — runs before every @Test, handing each a brand-new cart.
  @BeforeEach
  void setUp() {
    cart = new ShoppingCart();
  }

  @Test
  void addingOneItemCountsAsOne() {
    cart.add("Coffee", 450);
    assertEquals(1, cart.itemCount());
    assertEquals(450, cart.totalCents());
  }

  // Were the fixture NOT reset, this could see "Coffee" from the test above and
  // fail. The fresh cart from @BeforeEach keeps it isolated, so it passes.
  @Test
  void totalSumsAllItems() {
    cart.add("Tea", 300);
    cart.add("Muffin", 275);
    assertEquals(2, cart.itemCount());
    assertEquals(575, cart.totalCents());
  }
}

Want proof @BeforeEach runs before each test and not just once? LifecycleDemoTest prints each hook as it fires, so you see @BeforeEach / @AfterEach bracketing both tests while @BeforeAll / @AfterAll fire once:

@BeforeAll   — once, before all tests
  @BeforeEach  — fresh fixture for the next test
    >>> firstTest body
  @AfterEach   — tear the fixture down
  @BeforeEach  — fresh fixture for the next test
    >>> secondTest body
  @AfterEach   — tear the fixture down
@AfterAll    — once, after all tests
