Recursion / tail call

Recursion solves a problem by having a function call itself on a smaller piece of the same problem until it reaches a base case.
A tail call is a recursive call made as the final action of a function, which some languages can optimize to avoid growing the call stack.
JavaScript supports recursion directly, but tail call optimization is only partial in practice, so deep recursion can still overflow the stack.

// JavaScript
function factorial(n) {
  if (n <= 1) return 1;
  return n * factorial(n - 1);
}

console.log(factorial(5)); // 120
