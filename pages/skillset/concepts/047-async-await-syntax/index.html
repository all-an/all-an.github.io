Async/await syntax

async and await let asynchronous code read more like direct sequential code.
In TypeScript, an async function returns a promise, and await pauses that function until another promise settles.
The main idea is better syntax for chaining asynchronous steps without deeply nested callbacks.

// TypeScript
function delay<T>(ms: number, value: T): Promise<T> {
  return new Promise(resolve => setTimeout(() => resolve(value), ms));
}

async function run() {
  // Start the first async step and wait here until its Promise resolves.
  const user = await delay(100, { name: "Ana" as string });

  // Only after the first result arrives does execution continue to this step.
  const message = await delay(100, `hello ${user.name}`);

  // This line runs last, after both awaited operations have completed.
  console.log(message);
}

run();
