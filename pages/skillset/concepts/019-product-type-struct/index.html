Product type / struct

A product type groups several values together into one compound value, where each field exists at the same time.
You can think of it as combining pieces with an “and”: a point has an x and a y, a user has a name and an email.
In Java, classes and records can model product types, and records are a compact way to declare immutable data carriers.

// Java
record Point(int x, int y) {}

Point p = new Point(4, 7);

System.out.println(p.x()); // 4
System.out.println(p.y()); // 7
