Variadic / spread / rest

Variadic functions accept a variable number of arguments instead of requiring a fixed arity.
Different languages expose this with different syntax, but the idea is the same: collect many arguments and treat them as one group.
In Java, varargs use ... and are received as an array inside the method body.

// Java
class Demo {
  static int sum(int... values) {
    int total = 0;
    for (int v : values) total += v;
    return total;
  }

  public static void main(String[] args) {
    System.out.println(sum(1, 2, 3)); // 6
  }
}
