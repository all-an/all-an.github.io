Pattern matching

Pattern matching selects behavior by comparing a value against a set of shapes, cases, or structural rules.
It goes beyond simple equality checks by letting code branch on type, form, and sometimes nested data in one expression.
This makes complex condition handling more declarative and easier to read than long chains of manual tests.

// Java
Object value = 42;
switch (value) {
  case Integer n -> System.out.println("int: " + n);
  default -> System.out.println("other");
}
