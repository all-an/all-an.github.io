Condition variable

A condition variable lets one thread wait until some shared-state condition becomes true and another thread signals it.
In Java, a Condition is created from a lock and works like a more explicit form of wait/notify.
The usual pattern is: lock, check in a loop, await, update state, signal.

// Java
import java.util.concurrent.locks.Condition;
import java.util.concurrent.locks.ReentrantLock;

ReentrantLock lock = new ReentrantLock();
Condition ready = lock.newCondition();
boolean[] done = {false};

Thread waiter = Thread.startVirtualThread(() -> {
  lock.lock();
  try {
    while (!done[0]) ready.await();
    System.out.println("ready");
  } catch (InterruptedException ignored) {
  } finally {
    lock.unlock();
  }
});

lock.lock();
try {
  done[0] = true;
  ready.signalAll();
} finally {
  lock.unlock();
}

waiter.join();
