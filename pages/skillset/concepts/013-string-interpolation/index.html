String / interpolation

String interpolation builds text by embedding values directly inside a string literal instead of concatenating pieces manually.
It usually reads better because the final output format stays visible in one place.
In JavaScript, template literals use backticks and expressions inside ${...}.

// JavaScript
const name = "Allan";
const score = 42;
const message = `${name} scored ${score} points`;

console.log(message); // "Allan scored 42 points"
