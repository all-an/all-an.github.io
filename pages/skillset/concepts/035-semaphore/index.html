Semaphore

A semaphore controls access to a limited number of permits instead of allowing only a single owner like a mutex.
Java's Semaphore is often used to limit how many threads may use a resource at once.
Each acquire() takes a permit and each release() gives one back.

// Java
import java.util.concurrent.Semaphore;

Semaphore permits = new Semaphore(2);

try {
  permits.acquire();
  try {
    System.out.println("inside limited section");
  } finally {
    permits.release();
  }
} catch (InterruptedException ignored) {
}
