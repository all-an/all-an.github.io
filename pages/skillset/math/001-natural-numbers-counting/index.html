Natural numbers & counting

Arithmetic

The idea

The natural numbers are the counting numbers: 1, 2, 3, 4, … (many texts start at 0). They are the first numbers humans ever used — one tally mark per sheep — and every other kind of number is built on top of them.

Counting works because of one simple rule: every natural number has a successor, the number that comes right after it. Start at 1, take the successor to get 2, again to get 3, and you never run out. That single "and one more" step is the whole engine of counting.

Peano's axioms — counting, made precise

In 1889 Giuseppe Peano captured the natural numbers with a tiny set of rules. Informally:

  • 0 is a natural number (the starting point).
  • Every natural number n has a successor S(n), which is also a natural number.
  • 0 is not the successor of any number — nothing comes before the start.
  • If two numbers have the same successor, they are the same number.
  • Induction: if a property holds for 0 and survives the successor step, it holds for every natural number.
S(n) = n + 1   →   1, S(1)=2, S(2)=3, S(3)=4, …

Place value — writing big numbers with few symbols

We only have ten digits (0–9), yet we can write any natural number. The trick is place value: a digit's worth depends on its position, each place being ten times the one to its right.

3 7 4 = 3×100 + 7×10 + 4×1 = 300 + 70 + 4

This is the base-10 (decimal) system. The same idea with a different base powers computers (base-2, binary).
