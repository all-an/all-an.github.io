Reverse a string

Beginner String

Given a string, return a new string with its characters in the opposite order — for example "hello" becomes "olleh". It is the classic warm-up for string indexing and the two-pointer technique.

Try it

↓

Approach

Most languages ship a one-liner, but interviews usually want the manual version: scan the string from left to right and prepend each character onto a growing result — so the first character ends up last and the last ends up first — and reject a null input by throwing an exception.

Java

import java.util.Objects;

// Approach 1 — built-in: StringBuilder.reverse()
static String reverse(String s) {
    return new StringBuilder(s).reverse().toString();
}

// Approach 2 — manual: scan left to right, prepending each character.
static String reverseAString(String stringToReverse) {
    String reversedString = "";
    if (Objects.nonNull(stringToReverse)) {
        for (int i = 0; i < stringToReverse.length(); i++) {
            char tempChar = stringToReverse.charAt(i);
            reversedString = tempChar + reversedString; // prepend reverses order
        }
    } else {
        throw new IllegalArgumentException("String cannot be null");
    }
    return reversedString;
}

JavaScript

// Approach 1 — built-in: spread by code point, then reverse and join.
// Using [...s] (not s.split('')) keeps emoji and accented characters intact.
function reverse(s) {
  return [...s].reverse().join('');
}

// Approach 2 — manual: scan left to right, prepending each character
function reverseAString(stringToReverse) {
  if (stringToReverse == null) {
    throw new Error('String cannot be null');
  }
  let reversedString = '';
  for (let i = 0; i < stringToReverse.length; i++) {
    reversedString = stringToReverse[i] + reversedString; // prepend reverses order
  }
  return reversedString;
}

Complexity

  • Time O(n) with a StringBuilder — each character handled once
  • Space O(n) — a new string of the same length

The prepend version above reads cleanly, but because strings are immutable each tempChar + reversedString copies the whole result built so far, making it O(n²). Reach for StringBuilder in Java or an array join in JavaScript when the input can be large.
