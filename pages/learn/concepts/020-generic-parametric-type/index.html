Generic / parametric type

A generic type is written once but can work with many element types through a type parameter.
It improves reuse while still preserving type checking, so the compiler can enforce which values are allowed.
In Java, generics use angle brackets like <T> on classes, methods, and interfaces.

// Java
class Box<T> {
  T value;

  Box(T value) {
    this.value = value;
  }
}

Box<String> box = new Box<>("hello");
System.out.println(box.value); // hello
