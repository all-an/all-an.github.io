Type alias

A type alias gives an existing type a new name, usually to make code shorter or express intent more clearly.
It does not create a brand-new runtime structure by itself; it mostly helps the type system and the reader.
Java does not have true type aliases, but TypeScript does with the type keyword.

// TypeScript
type UserId = string;
type User = {
  id: UserId;
  name: string;
};

const user: User = {
  id: "u_123",
  name: "Allan",
};
