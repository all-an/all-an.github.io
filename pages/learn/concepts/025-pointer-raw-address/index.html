Pointer / raw address

A pointer is a value that refers directly to a memory location instead of storing the data itself.
Low-level languages expose raw addresses directly, while Java normally hides them behind references and safety checks.
With the Foreign Function & Memory API, Java can still work with native memory and reveal its address when needed.

// Java
import java.lang.foreign.Arena;
import java.lang.foreign.MemorySegment;
import java.lang.foreign.ValueLayout;

try (Arena arena = Arena.ofConfined()) {
  MemorySegment number = arena.allocate(ValueLayout.JAVA_INT);
  number.set(ValueLayout.JAVA_INT, 0, 42);

  long address = number.address(); // This raw memory address is the pointer value.
  int value = number.get(ValueLayout.JAVA_INT, 0);

  System.out.println("0x" + Long.toHexString(address));
  System.out.println(value); // Reads the int stored at that address: 42
}
